Detailed Description

tw_lp Holds our state for the LP, including the lptype and a pointer to the user's current state. The lptype is copied into the tw_lp in order to save the extra memory load that would otherwise be required (if we stored a pointer).

Specific PE's service specific LPs, each PE has a linked list of the LPs it services, this list is made through the pe_next field of the tw_lp structure.
